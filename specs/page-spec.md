PROJECT: Profile Card

WHAT IT IS:
A single-page card introducing a software developer to potential employers displayed on a personal portfolio homepage

MAIN PARTS:
- Profile photo (circular, top of card)
- Full name (heading)
- Professional title (subheading)
- 2-3 sentence bio
- "Hire Me" call-to-action button
- Social media icon links (LinkedIn, GitHub, Twitter)

STYLE RULES:
 Card container: 400px width, centered on page
- Background: #2a5efa
- Text: #FFFFFF
- Accent: #5e2afa
- Heading: 28px bold
- Body text: 16px regular
- Button: #2ac6fa, white text, rounded corners
- button width: 300px
- button height: auto
- button hover: darken
- button active state: drop shadow
- button focus state: outline 3px #5e2afa
- padding: 16px
- Photo: 120px diameter, 8px border
- Responsive: Stack vertically on mobile (<768px)
- Accessibility: ARIA labels on all interactive elements

SUCCESS CRITERIA:
- Card is visually centered on all screen sizes
- Button is clickable and leads to contact form
- All social links open in new tab
- Text is readable against dark background
- Loads in under 2 seconds
- Validates as semantic HTML5
